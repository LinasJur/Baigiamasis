# Baigiamasis
Norint paleisti programą, ja atsisiųsti iš git repozitorijos. 
Įsikėlus į Intelij arba bet kurią kitą programą, joje susibildinti back programą naudojant Maven.
Frontą pasileisti galima paspaudūs ant package.json dešinį mygtuką, atsiras meniu, jo apačioje pasirinkti Show npm scripts. 
Atsiradūs papildomai lentejėje, joje paspausti build, tada run. Taip turėtų atsidaryti naršyklėje appsas.
